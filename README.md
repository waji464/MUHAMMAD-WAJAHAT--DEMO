## Project Overview
The Light House School System is a web-based application designed to manage the administrative and academic functions of a school. It provides a user-friendly interface for students, teachers, and administrators to access important information and perform various tasks efficiently.
## Features 
- Student Management: Allows administrators to manage student records, including enrollment, attendance, and grades.
- Teacher Management: Enables administrators to manage teacher profiles, schedules, and assignments.
- Course Management: Provides tools for creating and managing courses, including syllabus, assignments, and assessments.
- Communication: Facilitates communication between students, teachers, and administrators through messaging and notifications.
## Usage
To use the Light House School System, follow these steps:   
1. Clone the repository to your local machine using the command: `git clone
    https://github.com/MUHAMMAD-WAJAHAT--DEMO/light-house-school-system.git`
2. Navigate to the project directory: `cd light-house-school-system`
3. Install the required dependencies: `npm install`
4. Start the application: `npm start`       
5. Access the application in your web browser at `http://localhost:3000`
